# 📞Customer Churn Prediction and Segmentation

## 🚀Project Overview
This project analyzes customer churn in a telecom company using Machine Learning techniques. The goal is to identify customers who are likely to leave the service and segment customers into meaningful groups for targeted retention strategies.

## ✨Features
* Exploratory Data Analysis (EDA)
* Data Cleaning and Preprocessing
* Feature Engineering
* Customer Churn Prediction
* Customer Segmentation using K-Means Clustering
* Model Evaluation and Comparison
* ROC Curve and Confusion Matrix Analysis

## 🧠Machine Learning Models Used
* Logistic Regression
* Random Forest Classifier
* K-Nearest Neighbors (KNN)
* K-Means Clustering

## 📂Dataset
The project uses the Telco Customer Churn dataset containing customer demographics, account information, services subscribed, and churn status.
Kaggle: Telco Customer Churn Dataset
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## 🔄Workflow

1. Data Loading and Exploration
2. Data Cleaning
3. Feature Encoding
4. Train-Test Split
5. Model Training and Evaluation
6. Churn Probability Prediction
7. Customer Segmentation
8. Performance Analysis

## 📊Results
The models were compared using multiple evaluation metrics to identify the best-performing approach for churn prediction. Customer segmentation was performed to classify customers into different risk and loyalty groups.
